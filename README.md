# csp.js

> Content Security Policy Generator

## Install

Install with `npm`

```bash
$ npm install csp.js
```

Or install with `yarn`

```bash
$ yarn add csp.js
```

```js
import CSP from 'csp.js';
```

Or umd builds are also available

```html
<script src="path/to/csp.js"></script>
```

Will expose the global variable to `window.CSP`.
